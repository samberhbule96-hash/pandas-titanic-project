# pandas-titanic-project
A data analysis project using Pandas to explore survival factors on the Titanic dataset

## Key Insights
| Factor | Observation |
| :--- | :--- |
| **Gender** | Females had a 74% survival rate. |
| **Class** | 1st Class passengers were prioritized. |

## How to Run
1. Clone the repo.
2. Install pandas: `pip install pandas`.
3. Run the notebook.
